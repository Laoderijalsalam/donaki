# Donaki

Landing page Donaki yang dibuat dengan HTML, CSS, dan JavaScript murni.

## Preview

![Preview halaman Donaki](draft/donaki-landing_1.webp)

## Fitur

- Navigasi halaman utama
- Hero section, kartu partner/member, dan informasi tentang Donaki
- Daftar partner
- Form newsletter
- Dialog login
- Tampilan responsif untuk layar yang lebih kecil

## Teknologi

- HTML5
- CSS3
- JavaScript

## Menjalankan proyek

1. Unduh atau clone repository ini.
2. Buka file `index.html` menggunakan browser.

Tidak diperlukan instalasi dependency atau server khusus.

## Struktur proyek

```text
.
├── index.html       # Struktur halaman
├── styles.css       # Tampilan dan responsivitas
├── main.js          # Interaksi halaman
└── draft/           # Aset gambar dan referensi desain
```
